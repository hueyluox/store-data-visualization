# 门店数据可视化系统

## 项目概述
基于门店运营数据（2024年6月至2025年10月）开发的数据可视化分析系统，涵盖3家门店（平安、侨城、双玺）的多维度业务指标监控与分析。

## 项目结构
```
门店数据可视化系统/
├── server.js                  # Node.js 服务器文件
├── pages/                     # 前端页面文件夹
│   ├── 门店指标走势分析.html   # 主要可视化分析页面
│   ├── dashboard.html         # 仪表板页面
│   └── ranking-table.html     # 排行榜页面
├── data/                      # 数据文件夹
│   └── 门店数据汇总 - 门店数据2024.6-2025.10.csv  # 原始数据文件
├── docs/                      # 文档文件夹
│   └── 门店数据可视化系统需求文档.md  # 详细需求文档
└── README.md                  # 项目说明文档（本文件）
```

## 功能特性

### 核心分析页面 (门店指标走势分析.html)
- **16个基础指标图表**: 总营收、确认收入、总客流、老客、各类转化率等
- **2个组合分析图表**: 老客分析、新客分析
- **门店切换功能**: 支持平安、侨城、双玺三家门店数据切换
- **标准化显示**: 统一的图表样式和交互体验

### 仪表板页面 (dashboard.html)
- 数据概览和关键指标展示
- 快速访问各分析模块

### 排行榜页面 (ranking-table.html)
- 门店间数据对比和排名
- 多维度数据排序功能

### 后端服务 (server.js)
- 静态文件服务
- API接口支持
- 跨域处理

## 技术栈

### 前端技术
- **HTML5 + CSS3**: 页面结构和样式
- **JavaScript (ES6+)**: 业务逻辑实现
- **Chart.js 3.9.1**: 图表可视化库
- **chartjs-plugin-datalabels**: 数据标签插件

### 后端技术
- **Node.js**: 服务器运行环境
- **Express.js**: Web框架
- **CORS**: 跨域资源共享

### 数据处理
- **CSV数据解析**: 原生JavaScript实现
- **数据可视化**: 多维度业务指标展示
- **响应式设计**: 适配不同屏幕尺寸

## 快速开始

### 1. 启动服务器
```bash
cd "/Users/oscar/Downloads/门店数据可视化系统"
node server.js
```

### 2. 访问应用
打开浏览器访问：
- 主页: http://localhost:3000
- 数据分析: http://localhost:3000/pages/门店指标走势分析.html
- 仪表板: http://localhost:3000/pages/dashboard.html
- 排行榜: http://localhost:3000/pages/ranking-table.html

### 3. 数据文件说明
数据文件位于 `data/` 文件夹中，包含：
- **时间范围**: 2024年6月至2025年10月
- **门店数量**: 3家（平安、侨城、双玺）
- **数据指标**: 29项业务指标
- **数据格式**: CSV格式，按门店、年份、月份组织

## 主要功能说明

### 基础指标监控
- 营收类指标：总营收、确认收入
- 客流类指标：总客流、老客
- 转化率指标：A/B/C/D各类转化率
- 服务效率指标：服务时长、人均服务时长
- 获客渠道指标：打鱼邀约、主动上门

### 组合分析
- **老客分析**: 老客总数及B/C/D类客户分布趋势
- **新客分析**: A类客户来源分析（美团点评、路过、老带新、其他）

### 交互功能
- 门店快速切换
- 图表悬浮提示
- 数据标签显示
- 参考线对比
