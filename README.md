# 门店数据可视化系统

## 项目说明
本系统包含两个数据可视化页面：

1. **门店指标走势分析** - 门店业务数据的趋势分析图表
2. **护理师雷达图** - 护理师绩效表现的雷达图分析

## 技术栈
- HTML5 + CSS3 + JavaScript
- Chart.js 图表库
- Chart.js Plugin DataLabels 插件

## 在线访问
通过GitHub Pages提供的在线访问链接（在启用Pages后可用）

## 本地运行
1. 下载项目文件
2. 使用HTTP服务器启动（推荐Python3）：
   ```bash
   python3 -m http.server 8080
   ```
3. 在浏览器中访问相应的HTML文件
